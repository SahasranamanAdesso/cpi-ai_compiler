## 6. Known Limitations

### Version 1.0

**Component Library**
- Only 3 components supported (HTTPS Sender, HTTP Receiver, Content Modifier)
- No Router, Groovy, Data Store, or other enterprise components

**Configuration**
- HTTPS Sender URL path is hardcoded (`/hello`)
- HTTP Receiver method is hardcoded (POST)
- No dynamic adapter configuration
- No environment-specific configurations

**Features**
- No deployment API (manual import required)
- No validation engine
- No testing framework
- Content Modifier only supports body (not headers/properties)

**All limitations are documented in:**
- CHANGELOG.md
- SUPPORTED_COMPONENTS.md
- ROADMAP.md
